# Factory Reallocation & Shipping Optimization

## Features

- Lead Time Prediction
- Factory Reallocation
- Scenario Analysis
- Recommendation Engine
- Risk Analysis
- Interactive Dashboard

## Charts Included

1. Sales by Region
2. Profit by Factory
3. Lead Time Distribution
4. Ship Mode Distribution
5. Factory Utilization
6. Product Profitability

## Run

Create environment

```bash
python -m venv venv
```

Install

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
